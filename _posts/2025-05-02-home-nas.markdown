## Server Set Up

1) Install SAMBA
2) Configure UFW firewall
3) add/fix file permission/groups/username/etc.
4) configure samba
5) mount drive
6) try auto mount drive


Just bunch of dumps in no particular order....
### Find Driev UUID
```
lsblk
blkid /dev/sda1
sudo mount /dev/disk1 /mnt/disk-name/

```
### Edit FSTAB and add auto mount
```
blkid
nano /etc/fstab/

```
